## What BetAnalyticsPro Is

BetAnalyticsPro is a web application for football betting analysis that centers on match-by-match prediction review and market interpretation. Its workflow relies on Poisson-style modeling to present probability-oriented insights in a browser-friendly interface.

This project is intended for anyone who wants a simple way to examine football results through a data-informed workflow. Because it operates in the browser, it avoids a local installation process while still keeping the analysis tied to the version 2.2 release.
